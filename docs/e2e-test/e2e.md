
# 通常ログインモード
## timeline画面
- [ ] タイムラインが表示されること
- [ ] 無限スクロールで過去の投稿が読み込まれること
- [ ] FAB（＋ボタン）から新規投稿（テキストのみ）ができること
- [ ] FAB（＋ボタン）から新規投稿（画像付き）ができること
- [ ] 投稿に対して返信ができること
- [ ] 投稿の画像をタップして拡大表示できること
- [ ] 自分の投稿を削除できること
- [ ] ブックマークの登録・解除ができること
- [ ] 通報機能が動作すること

## map画面
- [ ] マップが表示され、登録済みピンが表示されること
- [ ] 現在地ボタンで現在地に移動すること
- [ ] ピンをタップして詳細ダイアログが表示されること
- [ ] 詳細ダイアログからタイムライン詳細へ遷移できること
- [ ] マップ上の任意の地点をタップしてピン作成ダイアログが表示されること
- [ ] チャットピンを作成できること（場所名、カテゴリ選択）
- [ ] イベントピンを作成できること（期間、詳細、URL指定）
- [ ] 画像付きのピンを作成できること

## profile画面
- [ ] 自分のプロフィール情報（アイコン、名前、自己紹介、URL）が正しく表示されること
- [ ] 「プロフィールを編集」ボタンが表示され、編集画面へ遷移できること
- [ ] プロフィール編集画面で画像、ニックネーム、自己紹介、URLを更新できること
- [ ] 更新後、プロフィール画面に最新の情報が反映されていること
- [ ] バリデーションエラー（ニックネーム未入力など）が正しく表示されること
- [ ] 他ユーザーのプロフィールを閲覧できること（編集ボタンが表示されないこと）

## inbox画面
- [ ] 通知メッセージ一覧が表示されること
- [ ] メッセージをタップして既読にできること
- [ ] 「すべて既読にする」ボタンで一括既読にできること
- [ ] メッセージを削除できること
- [ ] 無限スクロールで過去のメッセージが読み込まれること
- [ ] メッセージがない場合、「メッセージはありません」と表示されること

## calendar画面
- [ ] カレンダーが表示されること
- [ ] 特定の日付を選択して、その日の投稿一覧が表示されること
- [ ] ドラッグ等で期間を選択して、期間内の投稿一覧が表示されること
- [ ] 選択期間の投稿に対して返信ができること
- [ ] 選択期間に投稿がない場合、その旨が表示されること

# ゲストモード
## timeline画面
- [ ] タイムラインの投稿一覧が閲覧できること
- [ ] 新規投稿や返信など、ログインが必要なアクションを行った際にログインプロンプトが表示される、または制限されること

## map画面
- [ ] マップが表示され、ピンが閲覧できること
- [ ] ピンの詳細情報が閲覧できること
- [ ] マップをタップしてピンを作成しようとすると、ログインプロンプトが表示されること

## profile画面
- [ ] プロフィール画面にアクセスすると「ログインが必要です」というプロンプトが表示されること
- [ ] ログインボタンからログイン画面へ遷移できること

## inbox画面
- [ ] 受信トレイにアクセスすると「ログインが必要です」というプロンプトが表示されること

## calendar画面
- [ ] カレンダーが表示され、投稿の検索・閲覧ができること
- [ ] 返信などのアクションを行おうとした際に制限される、またはログインプロンプトが表示されること
